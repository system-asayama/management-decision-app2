# Excelファイル分析結果

## ファイル名
簡易新エンジン3財務管理シミュレーション（個別計画あり）データあり20230301）.xlsx

## 概要
このExcelファイルは、企業の財務管理と経営シミュレーションを行うための包括的なツールです。49個のシートで構成されており、以下の主要機能を提供します。

## 主要機能カテゴリ

### 1. データ入力（インプット）
- **経営管理過去データ（インプット）－人件費データ**: 部門別・雇用形態別の人件費データ
- **経営管理過去データ（インプットーFS）**: 財務諸表データ

### 2. 財務諸表組換え
- **組換え手順（PL）**: 損益計算書の組換え手順
- **組換え手順（BS）**: 貸借対照表の組換え手順
- **PLの組換え (1)(2)(3)**: 3期分の損益計算書組換え結果
- **BSの組換え (1)(2)(3)**: 3期分の貸借対照表組換え結果

### 3. 経営分析（4つの視点）
#### 3.1 成長力分析（1成長力）
- 売上高成長率
- 売上原価成長率
- 付加価値成長率
- 人件費成長率
- 役員報酬成長率
- 資本再成費成長率
- 研究開発費成長率

#### 3.2 収益力分析（2収益力）
- 総資本経常利益率
- 自己資本経常利益率
- 経営資本営業利益率
- 売上高経常利益率
- 各種回転率

#### 3.3 資金力分析（3資金力）
- 自己調達率（自己資本比率）
- 金融調達率
- 引当調達率
- 信用調達率
- 借入金依存率
- 担保余力
- 金融負担率

#### 3.4 生産力分析（4生産力）
- 総資本付加価値率
- 付加価値労働生産性
- 利益生産性
- 従業員一人当り売上高

### 4. 高度な分析機能
- **収益力-損益分岐点分析**: 固定費・変動費分析、損益分岐点売上高
- **収益力-最小二乗法**: 費用の固変分解
- **収益力-内部留保シミュレーション**: 内部留保の推移予測
- **資金力-貢献度分析**: 資金増減要因分析
- **資金力-借入金許容限度額分析**: 借入可能額の計算
- **生産力-差額原価収益分析**: 意思決定のための差額分析

### 5. 総合経営計画
- **総合経営計画-予算管理**: 予算策定と管理
- **連続F実績・予算**: 複数期の実績と予算の比較
- **初年度/2年度/3年度シミュレーション**: 3期分の経営シミュレーション
- **成長力/収益力/資金力/生産力（シミュレーション版）**: 計画値での経営分析

### 6. 個別経営計画
- **労務費管理計画（3期分）**: 人件費の詳細計画
- **設備投資計画**: 設備投資の計画と管理
- **主要運転資金計画**: 運転資金の計画
- **資金繰り計画（3期分）**: 月次資金繰り表
- **資金調達返済計画**: 借入金の調達と返済計画

## データベース設計への示唆

### 必要なテーブル

#### 1. 企業マスタ
- 企業ID、企業名、設立年月日、業種、従業員数など

#### 2. 財務諸表データ
- **損益計算書（PL）**
  - 売上高、売上原価、売上総利益
  - 販売費及び一般管理費（詳細項目）
  - 営業利益、経常利益、税引前当期純利益、当期純利益
  
- **貸借対照表（BS）**
  - 流動資産（現金預金、売掛金、棚卸資産など）
  - 固定資産（有形固定資産、無形固定資産、投資その他の資産）
  - 流動負債（買掛金、短期借入金など）
  - 固定負債（長期借入金など）
  - 純資産（資本金、利益剰余金など）

#### 3. 人件費データ
- 部門別、雇用形態別（常勤/臨時）の人件費
- 役員報酬

#### 4. 経営指標マスタ
- 指標名、計算式、目標値、評価基準

#### 5. シミュレーションデータ
- シミュレーションID、期間、前提条件
- 計画値（売上高、費用、投資など）

#### 6. 個別計画データ
- 労務費計画
- 設備投資計画
- 資金繰り計画
- 借入金計画

## 実装の優先順位

### Phase 1: 基本機能
1. 企業マスタ管理
2. 財務諸表データ入力（PL/BS）
3. 財務諸表組換え機能
4. 基本的な経営分析（4つの視点）

### Phase 2: 高度な分析
1. 損益分岐点分析
2. 差額原価収益分析
3. 借入金許容限度額分析

### Phase 3: シミュレーション
1. 複数年度シミュレーション
2. 個別計画（労務費、設備投資、資金繰り）
3. 総合経営計画
